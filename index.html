Model Architectures

Hybrid CNNâ€“SVM: A CNN feature extractor (several conv blocks + global average pooling) trained on MRI images. Extracted features are fed to an SVM (RBF kernel or linear, cross-validated) for final classification. This approach often reduces overfitting and speeds up training compared to end-to-end deep fully-connected classifiers. 

CNN feature extractor (example)

Conv(32,3x3) -> ReLU -> BatchNorm -> MaxPool

Conv(64,3x3) -> ReLU -> BatchNorm -> MaxPool

Conv(128,3x3) -> ReLU -> BatchNorm -> GlobalAveragePool

Dense(256) -> ReLU -> Dropout(0.4)

--> Feature vector (size 256) stored and fed to SVM



SVM classifier

- Kernel: RBF or linear

- Hyperparams: C (1.0), gamma='scale'

ResNet Benchmark

Standard ResNet-50 fine-tuned on the same training split. We extract top-1 accuracy, per-class recall, precision and compute inference time (ms) on CPU and GPU.
